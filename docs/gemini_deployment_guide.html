Gemini AI Deployment Guide

Complete step-by-step instructions for Google Gemini AI integration

1 Generate Google AI API Key

Google AI Studio

Go to Google AI Studio and sign in with your Google Workspace or Personal account.

Create API Key

  1. Click "Get API key" in the left sidebar.
  2. Click the blue "Create API key" button.
  3. Select "Create API key in new project".

Copy the key immediately. It starts with AIzaSy....

2 Enable Advanced Models Required for Images/Video

While text generation is usually free (with rate limits), image and video generation (Imagen/Veo) often require a linked billing project.

Link Billing Account

In Google AI Studio, click on Settings (Gear Icon) at the bottom left.

Under Billing, ensure you have a project with an active billing account linked. This enables the Pay-as-you-go tier, which is required for Imagen 3.x and Veo models.

Tip: Even with a billing account, text generation remains free up to a certain limit on the Flash models.

3 Harvest Admin Integration

Update Configuration

Log in to your Harvest Admin panel and navigate to Configuration > API Keys.

Find the Gemini API Key field and paste your AIzaSy... key.

Click Save to persist your settings.

Test Connection

Click the "Test Connection" button next to the Gemini key field. You should see a success toast confirming the connection.

4 Active AI Features

The Harvest Admin uses Gemini for several high-impact features:

Smart Product Naming

When you upload a product image, click the Magic Wand icon. Gemini 1.5 Flash (Vision) will analyze the image and generate appetizing titles and descriptions.

Image Enhancement

Use the "Enhance with Gemini" feature to turn basic snapshots into professional studio-quality photos using Google Imagen 3.0.

Security Tip: Your API key is stored securely in config.json on your server. Never share this key or expose it in public repositories.
